# Music Playlist Management System

A console-based Java application to manage and navigate a music playlist.

## Features
• Add songs to albums
• Create playlists from multiple albums
• Play next, previous, and replay songs
• Delete current song
• Input validation and edge case handling

## Technologies Used
• Java
• OOP Principles
• Java Collections (ArrayList, LinkedList)
• ListIterator

## Future Enhancements
- Persistence (file or database)
- Shuffle and repeat modes
- GUI support
